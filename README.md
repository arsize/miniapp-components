## Dialog

弹窗提示，对普遍在移动开发中使用的弹窗组件进行了封装

##### 属性

- `size` 弹窗大小（normal：正常，large:大，small：小）
- `type` 弹窗样式类型
- `title` 弹窗标题
- `content` 弹窗内容
- `contentposition` 弹窗内容位置（居左或居中）
- `status` 是否显示蒙版
- `marsktap` 蒙版是否支持点击事件
- `foot` 是否有点击按钮

##### 使用

```javascript
let option = {
  status: true, //启用蒙版
  closeicon: true, //启用右上角关闭按钮
  content: `维护中，暂不可提供服务`,
  foot: [
    {
      text: "我知道了",
      cb: () => {},
    },
    {
      text: "确定",
      cb: () => {},
    },
  ],
};
emitter.emit("dialogstatus", option);
return;
```
